Detailed Description

Base class for Finite Mesh Variables version 0.

Handles the fundamental grid structure, memory allocation, coordinate mapping, basic BSSN evolution steps (RHS calculation), and Kreiss-Oliger dissipation. Does not implement specific boundary conditions or mesh refinement.

This class manages the computational grid, stores dynamical variables (BSSN, gauge, fluid, scalar fields), provides methods for finite differencing, basic time stepping support, constraint checking, and memory management. It forms the foundation for specific simulation setups and mesh refinement.
